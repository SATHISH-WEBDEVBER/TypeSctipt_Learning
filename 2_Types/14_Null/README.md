# Null

## Definition | வரையறை

Null represents an intentional absence of a value.

Null என்பது ஒரு value திட்டமிட்டு இல்லை என்பதை குறிக்கிறது.

---

## Why Use Null?

Null indicates that a value exists but is currently empty.

Value இருக்க வேண்டும், ஆனால் தற்போது இல்லை என்பதை குறிக்கிறது.

---

## Syntax

```typescript
let user: string | null = null;
```

---

## Real-World Example

```typescript
let selectedProduct: string | null = null;
```

No product has been selected yet.

இன்னும் எந்த product-மும் தேர்வு செய்யப்படவில்லை.

---

## Common Use Cases

* Selected User
* Selected Product
* Search Result
* Database Records

---

## Advantages

| Benefit                 | Description            |

## Summary

* Represents an empty value.
* Intentionally assigned.
* Common in application states.
* Useful for validations.
