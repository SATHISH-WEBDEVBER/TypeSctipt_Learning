# Data Types

## Overview | அறிமுகம்

Data types define the kind of value that can be stored in a variable.

Data Types என்பது ஒரு variable-ல் எந்த வகையான value சேமிக்கப்படலாம் என்பதை வரையறுக்கிறது.

---

## Purpose | நோக்கம்

Understanding data types is essential for writing type-safe and maintainable TypeScript applications.

Data Types-ஐ புரிந்துகொள்வது type-safe மற்றும் maintainable TypeScript applications உருவாக்குவதற்கு முக்கியமானது.

---

## Learning Objectives | கற்றல் நோக்கங்கள்

After completing this section, you will be able to:

* Understand TypeScript data types.
* Choose the appropriate datatype for a variable.
* Prevent common datatype-related mistakes.
* Write more reliable and maintainable code.

இந்த பகுதியை முடித்த பிறகு:

* TypeScript Data Types-ஐ புரிந்துகொள்ளலாம்.
* Variable-க்கு சரியான datatype-ஐ தேர்வு செய்யலாம்.
* Datatype தொடர்பான பொதுவான தவறுகளை தவிர்க்கலாம்.
* நம்பகமான மற்றும் பராமரிக்க எளிதான code எழுதலாம்.
