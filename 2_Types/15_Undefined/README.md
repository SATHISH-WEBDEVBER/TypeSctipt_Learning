# Undefined

## Definition | வரையறை

Undefined means a variable has been declared but has not been assigned a value.

Undefined என்பது variable உருவாக்கப்பட்டாலும் value assign செய்யப்படாத நிலையை குறிக்கிறது.

---

## Why Use Undefined?

Undefined indicates that a value has not been initialized.

Value இன்னும் initialize செய்யப்படவில்லை என்பதை குறிக்கிறது.

---

## Syntax

```typescript
let userName: string | undefined;
```

---

## Real-World Example

```typescript
let currentUser: string | undefined;
```

The user has not logged in yet.

பயனர் இன்னும் login செய்யவில்லை.

---

## Common Use Cases

* Optional Values
* Function Parameters
* API Responses
* User Inputs

---

## Advantages

| Benefit                       | Description        |
| ----------------------------- | ------------------ |
| Indicates Uninitialized State | Value not assigned |
| Useful for Optional Data      | Flexible handling  |
| Easy Validation               | Simple checks      |

---

## Summary

* Variable exists.
* No value assigned.
* Represents an uninitialized state.
* Common in optional values.
