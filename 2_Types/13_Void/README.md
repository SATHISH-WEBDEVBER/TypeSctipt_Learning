# Void

## Definition | வரையறை

The void type represents the absence of a return value.

Void datatype ஒரு function எந்த value-யும் return செய்யாது என்பதை குறிக்கிறது.

---

## Why Use Void?

Void makes function intentions clear.

Function எந்த value-யும் return செய்யாது என்பதை தெளிவாக காட்டுகிறது.

---

## Syntax

```typescript
function greet(): void {}
```

---

## Real-World Example

```typescript
function sendNotification(): void {
  console.log("Notification Sent");
}
```

---

## Advantages

| Benefit            | Description              |
| ------------------ | ------------------------ |
| Better Readability | Clear function behavior  |
| Type Safety        | Prevents invalid returns |
| Maintainable       | Easier to understand     |

---

## Summary

* Used with functions.
* Represents no return value.
* Improves readability.
* Provides type safety.
