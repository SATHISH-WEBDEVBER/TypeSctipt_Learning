
# TypeScript Learning Repository

## Overview | அறிமுகம்

### English

This repository documents my TypeScript learning journey through practical examples, structured notes, and organized documentation.

The goal is to learn TypeScript systematically while creating a reusable reference for developers and students.

### தமிழ்

இந்த Repository-யில் நான் TypeScript கற்றுக்கொள்ளும் பயணத்தை நடைமுறை உதாரணங்கள், ஒழுங்குபடுத்தப்பட்ட குறிப்புகள் மற்றும் ஆவணங்களின் மூலம் பதிவு செய்கிறேன்.

TypeScript-ஐ முறையாக கற்றுக்கொள்வதோடு, Developers மற்றும் Students பயன்படுத்தக்கூடிய ஒரு Reference Repository-யாக இதை உருவாக்குவதே இந்த Repository-யின் நோக்கம்.

### English

* Learn TypeScript step by step.
* Document concepts in a structured manner.
* Compare JavaScript and TypeScript implementations.

* JavaScript மற்றும் TypeScript செயல்பாடுகளை ஒப்பிடுவது.
* தெளிவான மற்றும் தொழில்முறை Documentation-ஐ பராமரிப்பது.
* எதிர்கால பயன்பாட்டிற்கான Knowledge Base உருவாக்குவது.

## Repository Structure | Repository அமைப்பு

### English

Each topic is maintained in a separate folder and contains all resources related to that concept.

### தமிழ்

ஒவ்வொரு Topic-மும் தனித்தனி Folder-இல் பராமரிக்கப்படுகிறது. அந்த Concept-க்கு தேவையான அனைத்து Resources-களும் அந்த Folder-இல் இருக்கும்.

```text
Topic Folder
├── Example.js
├── Example.ts
└── README.md
```

## Learning Methodology | கற்றல் முறை

### English

Every topic follows a consistent learning and documentation approach:

1. Study the concept.
2. Implement examples.
3. Compare JavaScript and TypeScript behavior.
4. Document findings in English and Tamil.
5. Maintain clean and organized notes.

### தமிழ்

ஒவ்வொரு Topic-மும் ஒரே மாதிரியான கற்றல் மற்றும் Documentation முறையை பின்பற்றுகிறது:

1. Concept-ஐ கற்றுக்கொள்வது.
2. உதாரணங்களை உருவாக்குவது.
3. JavaScript மற்றும் TypeScript செயல்பாடுகளை ஒப்பிடுவது.
4. கண்டறிந்த தகவல்களை English மற்றும் தமிழ் மொழிகளில் பதிவு செய்வது.
5. சுத்தமான மற்றும் ஒழுங்குபடுத்தப்பட்ட குறிப்புகளை பராமரிப்பது.
